♻️ EchoChic – Turning Waste into Worth

From Plastic to Purpose.

EchoChic is a social-impact-driven web platform that focuses on collecting waste plastic from nearby garbage areas, empowering youth through volunteering, and transforming plastic waste into creative, low-cost interior and lifestyle products for local and global markets.

This project combines environmental sustainability, youth empowerment, and e-commerce into one integrated system.

🌍 Live Project Link: https://echochicthetrashmakeover.netlify.app/
